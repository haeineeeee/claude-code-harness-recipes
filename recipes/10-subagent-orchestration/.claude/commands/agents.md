# /agents — 서브에이전트 사용 현황 조회

서브에이전트 실행 로그를 분석하여 사용 통계를 보여줍니다.

## 사용법

```
/agents [today|week|all]
```

## 동작

1. `.claude/logs/subagent-usage.log` 파일을 읽습니다.
2. 기간별 필터링을 적용합니다 (기본: today).
3. 에이전트 유형별 호출 횟수, 평균 결과 크기를 집계합니다.
4. 가장 자주 사용된 패턴과 개선 제안을 출력합니다.

## 출력 예시

```
📊 서브에이전트 사용 현황 (오늘)
┌──────────────────┬───────┬────────────┐
│ 유형             │ 횟수  │ 평균 결과  │
├──────────────────┼───────┼────────────┤
│ Explore          │ 5     │ 2,340자    │
│ Plan             │ 3     │ 4,120자    │
│ general-purpose  │ 8     │ 6,890자    │
└──────────────────┴───────┴────────────┘
```
