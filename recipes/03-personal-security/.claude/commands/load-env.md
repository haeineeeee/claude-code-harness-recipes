description: .env 에서 특정 KEY 1개만 안전하게 추출해 마스킹된 형태로 확인합니다
---

추출할 키 이름: $ARGUMENTS

주의: 전체 .env 파일을 출력하지 않습니다. 다음 단계를 수행하세요.

1. 프로젝트 루트의 `.env` 또는 `.env.local` 존재 확인 (없으면 에러 안내 후 종료)
2. `grep -E "^${ARGUMENTS}=" .env` 로 해당 키 라인만 추출
   - 여러 줄이 나오면 (.env.local 등 다른 파일까지 포함) 파일별로 구분해 출력
3. 값은 **마스킹 형태**로 표시:
   - 값 길이 ≤ 8: 전체 `***`
   - 값 길이 > 8: 앞 3자 + `***` + 뒤 3자
4. export 명령은 제안만 하고 사용자가 직접 실행하도록 안내:
   ```
   (추천) export $ARGUMENTS="<실제값>"
   ```
5. 전체 .env 내용은 절대 출력하지 않음 (훅이 2중으로 차단하지만 이중 확인)
