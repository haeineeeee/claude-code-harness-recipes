# /cmd-list — 등록된 Slash 명령 목록 조회

`.claude/commands/` 디렉토리의 모든 `.md` 파일을 읽어 등록된 Slash 명령 목록을 표시합니다.

## 절차

1. `.claude/commands/*.md` 파일 목록 조회
2. 각 파일의 첫 줄(# 제목)에서 명령 이름과 설명 추출
3. 아래 형식으로 출력:

| # | 명령 | 설명 | 파일 |
|---|---|---|---|
| 1 | /cmd-list | 등록 명령 목록 조회 | cmd-list.md |

총 등록 명령: X개 / 권장 상한 10개
