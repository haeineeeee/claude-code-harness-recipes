# /cmd-stats — Slash 명령 사용 통계

`.claude/command-usage.jsonl` 로그를 분석하여 명령별 사용 통계를 표시합니다.

## 절차

1. `.claude/command-usage.jsonl` 파일 존재 확인
2. 전체 기간 및 최근 7일 / 30일 사용 빈도 집계
3. 아래 형식으로 출력:

| 명령 | 전체 | 30일 | 7일 | 마지막 사용 |
|---|---|---|---|---|
| /cmd-list | 45 | 12 | 3 | 2026-04-23 |

## 인사이트

- 미사용 명령 (30일 0회) 목록
- 가장 자주 사용하는 Top 3
- 등록 대비 활성 사용 비율
