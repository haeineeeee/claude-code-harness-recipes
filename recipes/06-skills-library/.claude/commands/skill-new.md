# /skill-new — 새 스킬 등록

사용자에게 아래 정보를 순서대로 확인한 뒤 SKILL.md를 생성하라:

1. **스킬 이름** (영문 소문자 + 하이픈, 예: `code-review`)
2. **한 줄 설명** (이 스킬이 무엇을 하는지)
3. **트리거 키워드** 3개 이상 (사용자가 어떤 말을 할 때 이 스킬이 활성화되는지)
4. **동작 단계** (1~5단계로 구체적으로)
5. **주의사항** (선택)

수집이 끝나면:
- `.claude/skills/<이름>/SKILL.md` 파일을 표준 템플릿으로 생성
- 생성 후 확인 메시지 출력
- `/skill-list`로 전체 목록을 보여줌
