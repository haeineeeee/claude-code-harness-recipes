# /skill-stats — 스킬 사용 통계 리포트

`.claude/skill-usage.jsonl` 파일을 읽어 아래 리포트를 생성하라:

## 출력 형식

### 전체 요약
| 항목 | 값 |
|---|---|
| 총 호출 수 | N회 |
| 고유 스킬 수 | N개 |
| 기록 기간 | YYYY-MM-DD ~ YYYY-MM-DD |

### 스킬별 호출 빈도 (Top 10)
| 순위 | 스킬 | 호출 수 | 최근 사용 |
|---|---|---|---|

### 최근 7일 일별 추이
| 날짜 | 호출 수 | 가장 많이 쓴 스킬 |
|---|---|---|

파일이 없거나 비어있으면: "아직 스킬 사용 기록이 없습니다. 스킬을 사용하면 자동으로 기록됩니다."
